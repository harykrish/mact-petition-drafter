⏺ The fresh-context verification pattern — where a separate Opus call audits 
  work done by a prior call with zero shared memory — was the most important 
  architectural decision in this project. It works remarkably well: the verifier
   genuinely catches arithmetic errors the drafter makes, rather than 
  rubber-stamping its own output. But I had to build this separation myself. It 
  would be powerful if the API had first-class support for context isolation 
  between agents — a way to guarantee that agent B cannot see agent A's 
  reasoning, only the artifact it produced. Right now I'm relying on discipline,
   not enforcement.

  Building NyayaSetu reinforced my conviction that AI systems handling 
  high-stakes decisions — legal, medical — need architectural guarantees against
   self-validation, not just prompting. The fresh-context pattern is a crude 
  version of this, but I believe it should be a core primitive.

  I could reconcile facts deterministically downstream without parsing natural 
  language. One gap: when Opus extracts a numeric value with low confidence, I'd

   but the source is ambiguous" — beyond a single confidence float.
